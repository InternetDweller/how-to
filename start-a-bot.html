Для запуска бота в Telegram написать боту @BotFather с командой /newbot. Сначала задать боту название (может быть любым), потом username (должно заканчиваться словом bot и, возможно, не может начинаться с цифры). Токен из ответа скопировать. На сайте Metabot24 зайти в Настройки бота –> Каналы и создать новую привязку к Telegram. Вставить токен в нужное поле и включить опцию Использовать inline-кнопки. После этого нажать на кнопку Вебхук на созданной привязке.
Для запуска бота на веб-сайте на сайте Metabot24 зайти в Настройки бота –> Каналы и создать новую привязку к Metabot Widget. Включить опцию Использовать inline-кнопки, заполнить поля Заголовок и Подзаголовок. После этого код из раздела Код виджета полностью скопировать и вставить внутрь тега <script type="module"></script>.
